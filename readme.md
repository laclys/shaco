### Shaco

A static site generator(Base on Vite)

### Usage

```bash
shaco dev [dir]
## preview
shaco preview [dir]
## build
shaco build [dir]
```

### Dev

```bash
pnpm i

npm link

# compiler
pnpm run start

# test:unit
pnpm test:unit

# test:e2e
pnpm test:e2e

# release
pnpm release [--dry]

# run
shaco dev [dir]
## preview
shaco preview [dir]
## build
shaco build [dir]

```
